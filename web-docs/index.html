Installation

1. Copy Module to Your Project

# Copy the entire module directory
cp -r /path/to/word-export-module ./src/word-export-module

# Or use npm (if published)
npm install word-export-module

2. Install Dependencies

cd src/word-export-module
npm install

# Key dependencies installed:
# - docxtemplater: ^3.65.2 (DOCX processing)
# - pizzip: ^3.2.0 (ZIP archive handling)
# - jszip: ^3.10.1 (ZIP file creation)
# - lodash: ^4.17.21 (Utility functions)

3. Build Module

npm run build

# This compiles TypeScript and copies assets:
# - Compiles src/ to dist/
# - Copies config/, templates/, uploads/ to dist/
# - Generates type definitions

4. Set Up Persian Fonts (Important!)

# Linux (Ubuntu/Debian)
sudo apt-get install fonts-farsiweb

# Or manually install Persian fonts:
# - B Nazanin, IRANSans, B Titr
# - Place .ttf files in system fonts directory

System Requirements

  • Node.js: 14.0 or higher
  • npm: 6.0 or higher
  • TypeScript: 4.0 or higher (for TS projects)
  • Persian Fonts: B Nazanin, IRANSans recommended
  • Memory: 512MB RAM for processing large batches
